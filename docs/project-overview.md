# 基速基金量化分析平台 - 项目概览

## 📋 项目简介

**基速基金量化分析平台** (jisu_fund_analyzer) 是一个基于 Flutter 开发的专业桌面端基金分析工具，采用 Clean Architecture + BLoC 模式，专注于为个人和机构投资者提供专业、高效的基金数据分析和投资管理功能。

### 基本信息

- **项目名称**: 基速 (JiSu) 基金量化分析平台
- **版本**: v0.5.5
- **开发语言**: Dart 3.1.0+
- **UI框架**: Flutter 3.13.0+
- **架构模式**: Clean Architecture + BLoC Pattern + Repository Pattern
- **主要目标平台**: Windows 桌面应用
- **支持平台**: Android, Web (实验性)

### 项目愿景

打造一个**专业、智能、易用**的基金分析平台，帮助投资者：
- 📊 **智能分析**: 基于量化分析的基金评估和推荐
- 🎯 **精准决策**: 提供详细的数据支持和投资建议
- 🚀 **高效管理**: 一站式投资组合管理和跟踪
- 💡 **用户友好**: 直观的界面设计和智能交互体验

### 架构亮点

#### 🏗️ Clean Architecture
- **分层清晰**: 严格的关注点分离
- **依赖倒置**: 高层模块不依赖低层模块
- **易于测试**: 每层都可以独立测试
- **高度解耦**: 组件间松耦合设计

#### 🔄 BLoC状态管理
- **可预测性**: 明确的状态流转逻辑
- **可测试性**: 业务逻辑与UI完全分离
- **可维护性**: 清晰的状态管理模式
- **可扩展性**: 易于添加新的状态管理需求

#### 💾 三级缓存系统
- **L1缓存**: 内存缓存 (毫秒级访问)
- **L2缓存**: Hive本地缓存 (快速持久化)
- **L3缓存**: PostgreSQL企业级缓存
- **智能策略**: 基于访问频率的自动缓存优化
